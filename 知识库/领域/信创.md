# 信创（信息技术应用创新）

> 一句话：用国产软硬件替代国外产品，实现核心技术自主可控，防范供应链断供风险。

## 概述

信创（信息技术应用创新）是国家信息安全战略的核心举措，要求关键信息基础设施和重要信息系统优先采用国产化技术和产品。荣成市目前正在同步推进 **三个** 国产化适配改造项目：

| 项目 | 预算 | 工期 | 负责单位 |
|------|------|------|----------|
| 数字荣成平台国产化适配改造 | 49.50 万 | 150 天 | [[荣成大数据中心]] |
| 集成化应急管理指挥系统国产化改造 | 37.00 万 | 120 天 | 荣成市应急管理局 |

## 标准国产化技术栈（荣成方案）

### 操作系统
**银河麒麟高级服务器操作系统 V10 SP3**
- 公安部等保四级认证
- 支持鲲鹏 920、飞腾 2000+ 等国产 CPU
- 基于 Linux 内核自主研发，源代码自主可控

### 数据库
**达梦数据库 DM8**
- 国产数据库市场占有率第一
- 公安部等保三级认证
- 兼容 Oracle 语法，迁移成本低
- 支持双机热备、读写分离等高可用架构

### 中间件
**东方通应用服务器 TongWeb V7.0**
- 国产中间件市场占有率第一
- 支持 Java EE 8 规范
- 兼容 Spring Boot / Spring Cloud / MyBatis 等主流框架

## 改造范围

### 数字荣成平台（大屏端+移动端）
- **大屏端**: 六大模块（数字惠民、经济运行、城市管理、民生诉求、物联感知、数字底座）百余个子功能
- **移动端**: 人/产/城/境/管/信六大领域 211 项数据指标
- **改造内容**: MySQL 5.7 → 达梦 DM8 · Windows → 银河麒麟 V10 · WebSphere/Tomcat → 东方通 TongWeb · Vue.js 2.x + Spring Boot 2.x 代码适配

### 集成化应急管理指挥系统（应急管理局）

- **五大模块**: 应急综合管理门户、安全生产监管、防汛抗旱指挥、森林防火监测、**应急指挥联合调度**
- **改造内容**: MySQL 5.7 → 达梦 DM8 · Tomcat → 东方通 TongWeb
- **数据规模**: 核心数据库约 30GB · GIS 空间数据约 5GB · 日志每日 ~2GB
- **项目预算**: 37.00 万元（120 天）

**应急指挥设备适配（通信层）**：

| 设备 | 用途 | 改造要点 |
|------|------|----------|
| 单兵设备 | 现场救援人员便携通信终端 | 替换国外通信协议栈，接入国产加密通道 |
| 无人机 | 空中侦察、物资投送 | 飞控数据对接、视频流传输、国产 SDK 集成 |
| 布控球 | 临时布控区域视频监控 | 云台控制协议适配、H.264/H.265 解码、NVR 对接 |
| 三防终端 | 恶劣环境数据采集 | 数据采集接口适配、GPS 定位对接、离线数据同步 |
| 图传设备 | 应急现场视频无线传输 | 编解码适配、传输链路调整、多路并发处理 |

## 数据迁移要点

- **工具**: Kettle 或 DataX（ETL）
- **策略**: 全量迁移 + 增量同步 + 双轨运行
- **校验**: 记录数校验 + 字段和校验 + MD5 校验
- **安全**: 国密 SM2/SM3/SM4 算法加密传输和存储
- **回退**: 2 小时内可恢复至原系统

## 荣成市信创生态

本次改造选用的国产产品均为市场占有率第一的头部产品，形成了可复制的"荣成方案"：

```
硬件层：鲲鹏/飞腾 CPU → 操作系统层：银河麒麟 V10
                                    ↓
中间件层：东方通 TongWeb ← → 数据库层：达梦 DM8
                                    ↓
应用层：Vue.js + Spring Boot（适配后）
```
