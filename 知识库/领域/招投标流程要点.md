# 招投标流程要点

> 一句话：招投标本质是"在规则范围内，用最好的方案和最合理的价格打动评标专家"。

## 关键阶段

1. **招标文件分析** — 找出隐性要求、评分点、废标项
2. **技术方案编写** — 贴合评分标准，突出差异化优势
3. **商务应答** — 逐条响应，不遗漏不偏离
4. **报价策略** — 平衡中标概率和项目利润
5. **投标文件封装** — 格式、盖章、密封，零失误

## 常见废标原因

- 未按要求盖章或签字
- 投标有效期不足
- 资质证书过期
- 技术方案与招标文件实质性偏离
- 报价超过预算

## 实战案例

### "爱山东"APP 荣成分厅建设运营项目（2023）

- **采购人**: [[荣成大数据中心]]
- **招标代理**: 山东东成建设咨询有限公司
- **采购方式**: 公开招标（全流程电子化）
- **项目编号**: SDGP371082000202202000180
- **预算金额**: 115 万元 → **中标金额**: 113.8 万元（报价策略：接近预算但合理让利）
- **评标方式**: 综合评分法（详见招标文件第四章）
- **投标保证金**: 无（政府采购改革后取消）
- **联合体投标**: 不接受
- **获取文件**: 威海市政府采购电子交易系统（网上获取）
- **开标地点**: 荣成市河阳东路 81 号 1 号楼（远程解密，不到现场）
- **中标供应商**: [[中国广电山东网络有限公司]]
- **付款方式**: 验收后 40% → 1 年后 30% → 2 年后 20% → 3 年后 10%（质保期 3 年，付款周期长）
- **特别要求**: 供应商须在荣成市设立固定项目部地点

**关键启示**:
1. 政府采购全流程电子化已是常态（威海市已全面推行）
2. 付款周期长达 3 年，需评估现金流压力
3. 知识产权归属甲方（源代码完全开放）
4. 系统需适配信创设备——这是政府项目的新趋势
