# Obsidian 主题与界面美化

> 一句话：Obsidian 外观可定制度极高，从原生设置到第三方主题再到 CSS 手写，三层递进，丰俭由人。

## 三层定制体系

```
原生外观设置 ──→ 插件 + 主题 ──→ CSS 手写定制
   (零门槛)        (低门槛)        (高自由度)
```

## 一、原生外观设置

### 缩放与字体
- **设置 → 外观 → 缩放比例**：调整图标等 UI 元素大小，字体大小建议保持默认
- **字体**：默认字体已足够好，也可在外观设置中更换界面/正文字体

### 工作区分区
- 将标签页拖拽到下方可创建新分区，拖回可合并
- 左侧功能区过窄时图标可能被遮挡，建议将面板都拖到下方，保留一个即可完整显示
- **堆叠标签页**（右键标签页 → 堆叠标签页）：横排标签变为纵向排列，长标题也能完整显示

### 界面元素显示/隐藏
| 选项 | 位置 | 效果 |
|------|------|------|
| 页内标题 | 设置 → 外观 | 隐藏笔记顶部的重复标题 |
| 标题栏 | 设置 → 外观 | 隐藏顶部标题栏 |
| 左侧功能区 | 设置 → 外观 | 隐藏后通过 Ctrl+P 命令面板访问所有功能 |
| 主题色 | 设置 → 外观 | 控制按钮、链接颜色 |

> 隐藏功能区后，Ctrl+P 命令面板包含了左侧所有按钮的功能。

## 二、插件 + 主题美化

### 核心组合：Style Settings + AnuPpuccin 主题

**安装步骤**：
1. 社区插件市场搜索安装 **Style Settings**
2. 外观 → 主题管理 → 搜索 **AnuPpuccin** → 下载使用
3. 安装进阶配色：Style Settings → AnuPpuccin → Colors → 点击链接下载 CSS 文件 → 放入外观 → CSS 代码片段 → 重载

**Style Settings 常用配置**：

| 配置项 | 位置 | 说明 |
|--------|------|------|
| 主题背景 | Colors | 根据亮/暗主题选择对应选项，需与外观设置中的主题匹配 |
| 颜色覆盖 | Colors | 若覆盖了原生主题色，关闭"主题色"即可恢复自定义颜色 |
| 工作区布局 | Workspace → Workspace Layout | 三种模式：默认 / Border（整体圆角边框） / Cards（独立卡片） |
| Border 参数 | Border Layout | 圆角半径、边距宽度 |
| Cards 参数 | Cards Layout | 阴影开关、工具栏加框、文件列表加框 |
| 标题颜色 | Heading | 可为 H1~H6 各级标题单独设置颜色 |
| 文字装饰 | Text Decoration | 粗体自动变红、斜体变绿、高亮加黄底 |
| 彩虹文件夹 | Workspace → Rainbow Folders | Float 或 Simple 样式，文件夹带彩色标识 |
| 状态栏浮动 | Workspace → Status Bar → Floating | 鼠标移上才显示，界面更简洁 |
| UI 元素隐藏 | Show Hide UI Elements | 标题栏自动弹出、隐藏滚动条/状态栏 |

### 辅助插件

| 插件 | 用途 |
|------|------|
| **Commander** | 自定义左侧边栏按钮，比原生工具栏丰富得多。可添加"在新标签页中打开光标处链接"等命令。不同设备图标可能不同 |
| **Hide** | 隐藏仓库名（Hide Vault Name）、隐藏标签栏（Hide Tab Bar）。隐藏仓库名后设置键会消失，可用 Commander 添加设置按钮或 Ctrl+, 呼出 |

## 三、CSS 定制

### 通用方法（不需要会写 CSS）

1. **Ctrl+Shift+I** 打开开发者工具（Dev Tools）
2. 用元素选择器（左上角箭头）点击想修改的界面元素
3. 右侧会定位到对应 HTML 代码
4. **截图代码**发给 AI（如 ChatGPT），描述需求 → AI 生成 CSS
5. 将 CSS 保存为 `.css` 文件 → 放入外观 → CSS 代码片段文件夹 → 勾选启用

> 关键技巧：只要截图包含组件名称，AI 生成的 CSS 会非常准确。

### 实用示例
- 日历插件默认尺寸太大 → CSS 缩小适配侧栏
- 任何界面元素不满意 → 定位 → AI 生成 → 应用

## 四、实用小技巧

### Excalidraw 新功能
| 功能 | 操作 | 说明 |
|------|------|------|
| Selection 模式 | Preference → Wrapped / Overlap | Wrapped 需完全框选才选中；Overlap 只需选中一部分 |
| Tool Lock | Preference 中开启 | 连续绘制同一图形；关闭则绘制一次后自动回到选择工具 |
| 查看模式 | 进入后无法编辑 | 防止误触；按 Alt+Up 退出；查看模式下点击问号可查看快捷键 |
| 对齐线 | 按住 Ctrl 移动元素 | 自动出现对齐参考线 |
| 等距分布 | 选中多个元素 → 顶部工具栏"等距分布" | 自动整齐排列，可组合成组 |

### 日历插件（Calendar）
- 安装后显示在右侧栏
- 点击日期创建日记 → 需先启用核心插件"日记"并指定日记存放目录
- 勾选"Show Week Number"显示周数 → 点击周数可创建周记
- 周记目录建议与日记相同

### 其他技巧
- **鼠标中键竖选**：按住中键拖动可竖选一列文字，适用于批量修改数字或字母
- **文件链接**：复制文件路径（`file:///...`）粘贴到笔记中，点击打开资源管理器对应目录
- **PDF 嵌入**：将 PDF 直接拖入笔记，自动显示嵌入视图，不影响笔记布局，可翻页，左侧有缩略图导航

## 核心理念

> "外观不是最重要的，内容才是。" —— 视频作者 Bastandern

外观定制的目标不是花哨，而是**让界面服务于内容**：清爽、高效、减少视觉干扰。
