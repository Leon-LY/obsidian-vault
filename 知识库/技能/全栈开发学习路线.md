# 全栈开发学习路线

> 一句话：从零到可部署全栈项目的一年学习路线，React+Next.js+Node.js+PostgreSQL 技术栈。

## 学习路线总览

### Phase 1：基础（1-2 月）

**HTML5** — 语义标签、表单、ARIA 无障碍、SEO 友好结构
**CSS3** — 盒模型、Flexbox、Grid、响应式媒体查询、过渡动画
**JavaScript ES6+** — let/const、箭头函数、Promise、async/await、DOM 操作、fetch、事件循环
**Git** — 克隆、分支、PR、解决冲突、提交信息规范

🎯 **产出**：用 GitHub Pages 部署一个静态交互页面（如待办清单）

### Phase 2：前端（3-5 月）

**核心框架**
- **React** — Hooks（useState/useEffect/useContext/自定义）、组件生命周期、props vs state、React Router
- **TypeScript**（必须）— 接口、泛型、强类型

**样式与组件**
- **Tailwind CSS** — 工具类优先，2025-2026 最主流
- 组件库：shadcn/ui、Chakra UI、Radix

**状态与数据**
- 全局状态：**Zustand**（轻量现代）或 Redux Toolkit
- 服务端状态：**TanStack Query**（原 React Query）

**工具链**
- **Vite** 构建 / **Next.js**（见 Phase 4）
- ESLint + Prettier

🎯 **产出**：调用公开 API 的响应式 SPA（天气面板、电影搜索）

### Phase 3：后端（6-8 月）

**运行环境与框架**
- **Node.js** + **Express**（或 Fastify）
- 路由、中间件、错误处理、RESTful API 设计

**数据库**
- **SQL**（PostgreSQL 或 MySQL）— JOIN、索引、范式化
- **NoSQL**（MongoDB）— 灵活 schema
- **ORM：Prisma**（强烈推荐）或 TypeORM

**认证与安全**
- Session vs JWT
- OAuth（Google/GitHub 登录）
- 密码哈希 bcrypt/argon2
- XSS、CSRF、SQL 注入防护

**测试**
- **Vitest** 单元测试
- **Supertest** API 端点测试

🎯 **产出**：带认证和 CRUD 的 REST API（笔记应用），连接 React 前端

### Phase 4：全栈整合（9-11 月）

**元框架**（前后端一体化）
- **Next.js** — 文件路由、Server Actions、API Routes、SSR/SSG — 事实标准
- **tRPC** 类型安全 API
- 或 **GraphQL**（Apollo/Urql）

**实战特性**
- 文件上传（AWS S3 / Cloudinary）
- 支付（Stripe）
- 实时更新（WebSocket / Socket.IO）

**性能**
- 缓存策略、图片优化、懒加载、Core Web Vitals

🎯 **产出**：一个小 SaaS 产品（如项目管理工具，含用户认证、团队协作、计费）

### Phase 5：DevOps 与进阶（12 月+）

**部署**
- **Vercel** 前端 / **Render** 或 **Railway** 或 **AWS** 后端

**CI/CD**
- **GitHub Actions** 自动测试和部署

**容器化**
- **Docker** 基础

**监控**
- **Sentry** 错误追踪
- **Datadog** / **Logtail** 日志

**进阶技能（初中级→高级）**
- 数据结构与算法（数组、哈希表、树、图、Big O）
- 系统设计基础
- 云平台基础（AWS EC2/S3/Lambda）
- Web 安全基础

🎯 **产出**：带 CI/CD 和容器化部署的完整项目

## 推荐技术栈（2025-2026）

| 层级 | 推荐技术 |
|------|----------|
| 前端 | React + TypeScript + Next.js |
| 样式 | Tailwind CSS |
| 后端 | Node.js + Express（或 Next.js API Routes）|
| 数据库 | PostgreSQL（主力）+ 可选 MongoDB |
| ORM | Prisma |
| 状态管理 | Zustand + TanStack Query |
| 部署 | Vercel / Render / AWS |
| CI/CD | GitHub Actions |
| 容器 | Docker |

## 趋势

- **TypeScript 已是必修课**
- **Next.js 成为 React 框架的默认选择**
- **AI 辅助编码**（Cursor / Copilot / Claude）已成为预期技能——但必须理解生成的代码
- **全栈 JavaScript（MERN/PERN）** 占主导，单一语言降低复杂度
- **T 型技能** — 全栈广度 + 1-2 个领域的深度
