# 并蒂星球

> **一句话**：中国首款双胞胎育儿伴侣——双轨同步记录、镜像对比曲线、AI 作息协同，填补中文市场空白。
>
> **创始人即用户**：Leon，一对龙凤胎（2022 年 7 月 7 日出生）的父亲。
>
> **阶段**：Phase 0 基础准备（本周启动）。

## 项目概况

| 维度 | 详情 |
|------|------|
| **定位** | 专为双胞胎/多胞胎家庭打造的 AI 育儿管家 |
| **产品形态** | 微信小程序（MVP）→ App |
| **技术栈** | uni-app (Vue3) + 腾讯云 CloudBase + DeepSeek |
| **MV工期** | 10 周（2026.07 ~ 2026.09） |
| **目标用户** | 0-3 岁双胞胎父母（存量 80~100 万对） |
| **商业模式** | 免费 + 高级 ¥79/半年 |
| **市场状态** | 中文市场零竞品，仅英文 Twiniversity |

| # | 功能 | 特点 | 来源 | 阶段 |
|:--:|------|------|------|:---:|
| 1 | 🏆 **萌芽日记** | 记录两个宝宝之间的关系发展史 | 妈妈原创 | MVP |
| 2 | 🏆 **就诊速查卡** | 一键导出医生真正需要的双胞胎对比摘要 | 🆕医生 | MVP |
| 3 | 🏆 **爸爸值班模式** | 妈妈出差→一键切换 SOP 清单 | 宝爸原创 | MVP |
| 4 | 🏆 **今天我做了什么** | 一句话记录爸爸的育儿贡献 | 宝爸原创 | MVP |
| 5 | 🏆 **异常事件日志** | 不用 AI 分析的电子便签纸，兜底早期预警 | 🆕医生 | MVP |
| 6 | **双生小报告** | 老师 10 秒记录→双胞胎互动瞬间推送家长 | 🆕老师 | P2 |

## 执行阶段

| 阶段 | 时间 | 目标 |
|------|------|------|
| **Phase 0** | 2026.07（1~2 周） | 环境搭建、曲线 Demo、20 篇核心内容 |
| **Phase 1** | 2026.07~09（10 周） | MVP：极简双轨记录 + 对比生长曲线 + 知识库 |
| **Phase 2** | 2026.10~12（3 月） | AI 作息引擎、AI 育儿顾问、社区、付费 |
| **Phase 3** | 2027.Q1+ | 独立 App + IoT + B2B |

**止损决策点**：Phase 1 上线后 3 个月，日活 < 50 或留存 < 30% 或 Leon 撑不住 → 暂停。
