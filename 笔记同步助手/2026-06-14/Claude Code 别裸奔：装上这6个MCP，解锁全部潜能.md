## Claude Code 别裸奔：六个MCP安装指南

Claude Code 本身很强，但如果不安装 MCP 就像在裸奔。真正让它干活，需要给它装上 MCP（模型上下文协议），相当于给 Claude Code 插上外挂。

### 什么是 MCP？

MCP（Model Context Protocol）是给 Claude Code 扩展能力的接口。通过安装不同的 MCP，Claude Code 可以访问浏览器、文件系统、数据库、GitHub 等外部资源。

### 六个必备 MCP

#### 1\. Playwright MCP

给 Claude 装上浏览器能力。它能够打开网页、点击按钮、查看页面、跑流程，还能帮你验收前端。以前你需要给 AI 截图，现在 AI 自己进浏览器看。

#### 2\. File System MCP

给 Claude 一块施工现场。它能够读文件、搜代码、改项目、整理目录。不需要你一段段复制，让 Claude 自己进项目里找。注意：不是全盘乱翻，你授权哪里它就在哪里开工。

#### 3\. Postgres MCP

给 Claude 接上数据库。它能看表结构、写 SQL、跑直读查询，帮你查数据、做分析、排问题。以前 AI 靠你描述，现在它能看数据现场。

#### 4\. GitHub MCP

把 GitHub 接入 Claude Code。它能够查看 Issue、PR、Actions、仓库代码、review 线索。不再需要在 GitHub 和聊天框之间来回横跳，一个人管理多仓库，这就是续命外挂。

#### 5\. Context 7 MCP

专治 AI 瞎写。API、React 变了、Next.js 也变了，文档一直在更新。别让 AI 凭记忆写，先让它查文档。Context 7 就是给 Claude 接一根最新文档管，减少幻觉、过期信息和报错。

#### 6\. Sequential Thinking

给 Claude 装上慢思考模式。面对复杂架构、难 bug、技术决策，别让 AI 秒回。让它拆步骤、做推理、回头检查。在追求快速答复之前，难题需要慢想。

### 总结

这六个 MCP 装上去，Claude Code 就不只是写代码了：它能进浏览器、能翻项目、能查数据库、能读新文档、还能慢下来推理。这才是 Claude Code 起飞的正确打开方式。
