## 一、原生外观设置

### 1\. 页面比例与缩放

打开设置 → 外观，调整缩放比例可改变图标等元素的大小，字体大小建议使用默认值。

![[笔记同步助手/images/ae5138cccd3974bdf35957a718367596_MD5.jpg|缩放比例设置]]

缩放比例设置

### 2\. 工作区分区与布局

默认三块面板。将标签页拖拽到下方可创建新分区，拖回可合并。左侧功能区较窄时图标可能被遮挡，建议将面板都拖到下方，保留一个即可完整显示。

开启“堆叠标签页”（右键标签页 → 堆叠标签页）可将横排标签变为纵向排列，长标题也能完整显示，点击即可切换。

![[笔记同步助手/images/5cefe09ab7971ae5f1e413fa34d28d8a_MD5.jpg|堆叠标签页效果]]

堆叠标签页效果

### 3\. 外观设置中的常用选项

-   **主题色：**对应按钮、选项、链接的颜色，可自由选择（如蓝色）。
-   **界面元素：**可关闭页内标题、上方标题栏、左侧功能区。隐藏功能区后，可通过 Ctrl+P 命令面板访问所有功能。
-   **字体：**默认字体可用，也可在外观设置中更换。

![[笔记同步助手/images/ab312f48586bde11b3dfac90d276d6ff_MD5.jpg|外观设置界面]]

外观设置界面

## 二、插件与主题美化

### 1\. 插件：Style Settings

在第三方插件中搜索安装“Style Settings”。

### 2\. 主题：AnuPpuccin（或其他兼容主题）

在外观 → 主题管理中搜索并下载使用。安装后界面颜色已发生变化。

若要更丰富的配色，需下载进阶版：在 Style Settings 设置页的 Colors 下点击链接下载 CSS 文件，然后在外观 → CSS 代码片段中导入并重载。

![[笔记同步助手/images/9e920f4a2d1fb19f5f61e2ec5ecb2626_MD5.jpg|Style Settings 插件与主题安装]]

Style Settings 插件与主题安装

### 3\. 使用 Style Settings 进行微调

按 Ctrl+P 输入“Style Settings”，选择“Show Style Settings View”将其显示在标签页中，方便实时预览。

-   **主题背景：**根据亮/暗主题选择对应选项，并确保外观设置中的主题与之一致。
-   **颜色覆盖：**若 Style Settings 修改后覆盖了原生设置的颜色，可在外观设置中关闭主题色，即可恢复自定义颜色。

![[笔记同步助手/images/5ce1fa9bfe4f980100bbba1b96b52c04_MD5.jpg|Style Settings 界面]]

Style Settings 界面

### 4\. 工作区布局：Border / Cards

在 Workspace 下可设置三种模式：默认、Border（整体边框圆角）、Cards（每块卡片独立边框）。Border 模式下可调节圆角半径与边距；Cards 模式可开启阴影，并决定工具栏、文件列表是否加框。

![[笔记同步助手/images/c2b6c2f9cd2fc9fec991ca28d8a63f70_MD5.jpg|Border 布局效果]]

Border 布局效果

### 5\. 显示/隐藏 UI 元素

在 Show Hide UI Elements 中，可设置标题栏自动弹出（需先在外观中开启标题栏显示）、隐藏滚动条、隐藏状态栏。状态栏有更好的控制方式：在 Workspace → Status Bar 中选择 Floating，鼠标移上才显示。

![[笔记同步助手/images/19db814f93bf9273ca908296455ad2d9_MD5.jpg|UI 元素隐藏设置]]

UI 元素隐藏设置

### 6\. 标题颜色与装饰

在 Heading 下开启“Enable Custom Heading Colors”，即可为各级标题单独设置颜色。在 Text Decoration 中开启后可设置粗体自动变红、斜体变绿、高亮加黄底。

![[笔记同步助手/images/c50ad4fc1d3e16e31278eca33cec9d37_MD5.jpg|标题颜色设置]]

标题颜色设置

### 7\. 彩色文件夹（Rainbow Folders）

在 Workspace → Rainbow Folders 中选择 Float 或 Simple 样式，并调整颜色参数，使文件夹带有彩色标识。

![[笔记同步助手/images/dcef7b2199717ab6e6058e06cdb7fd57_MD5.jpg|彩虹文件夹效果]]

彩虹文件夹效果

## 三、辅助插件：Commander 与 Hide

### 1\. Commander

安装后启用，在选项的“左侧边栏”中可添加命令按钮，比原生工具栏更丰富。例如添加“在新标签页中打开光标处链接”命令，保存后按钮会出现在侧栏。

注意：该按钮在不同设备（电脑/手机/平板）上的图标可能不同。

![[笔记同步助手/images/7dd034b0bb69f2e4a9f83f3348a0d3b1_MD5.jpg|Commander 添加命令]]

Commander 添加命令

### 2\. Hide 插件

主要用于隐藏仓库名（Hide Vault Name），可让界面更简洁。隐藏后设置键也会消失，可通过 Commander 添加设置按钮，或使用快捷键 Ctrl+逗号呼出设置。该插件还可隐藏标签栏（Hide Tab Bar）。

![[笔记同步助手/images/64f1fc326a50a5103433176effbfd0d7_MD5.jpg|隐藏仓库名效果]]

隐藏仓库名效果

## 四、CSS 定制

按 Ctrl+Shift+I 打开开发者工具（Dev Tools）。用元素选择器点击想修改的界面元素，右侧会定位到对应 HTML 代码。将代码截图交给 AI（如 ChatGPT），描述需求，AI 会生成准确的 CSS。

将 CSS 代码保存为 .css 文件，放入 Obsidian 外观 → CSS 代码片段文件夹，然后在设置中勾选启用。

例如，可让日历插件尺寸缩小以适应侧栏。

## 五、实用小技巧

### 1\. Excalidraw 新功能

-   **Selection 模式：**“Wrapped”需完全框选才选中；“Overlap”只需选中一部分即可。
-   **Tool Lock：**类似锁定键，开启后可连续绘制同一图形，关闭则绘制一次后自动回到选择工具。
-   **查看模式：**进入后无法编辑，防止误触；按 Alt+Up 退出。查看模式下点击问号可查看快捷键。
-   **对齐线：**按住 Ctrl 移动元素会出现自动对齐线。
-   **等距分布：**选中多个元素，点击顶部工具栏的分布图标（等距分布）即可自动整齐排列，还可组合成组。

![[笔记同步助手/images/2909aadd82c678f3d724fcd3c1a3ce15_MD5.jpg|Excalidraw 偏好设置]]

Excalidraw 偏好设置

![[笔记同步助手/images/1fc99bb978f20412fd1a045c7d9b6d81_MD5.jpg|Excalidraw 对齐与分布]]

Excalidraw 对齐与分布

### 2\. 日历插件（Calendar）

安装后显示在右侧。点击日期会创建日记，建议先启用核心插件中的“日记”功能，并在设置中指定日记存放目录。勾选“Show Week Number”可显示周数，点击周数可创建周记，同样需设置周记目录（建议与日记相同）。

![[笔记同步助手/images/44fa2b819c65da9939d99cc9daf509e7_MD5.jpg|日历插件设置]]

日历插件设置

### 3\. 鼠标中键竖选

按住鼠标中键拖动可选择一列文字，适用于批量修改数字或字母（如将所有数字改为777）。

![[笔记同步助手/images/5c2d388df56c600ffa663558e2753f59_MD5.jpg|中键竖选示例]]

中键竖选示例

### 4\. 文件链接与 PDF 嵌入

**文件链接：**直接复制文件路径（格式如 file:///...）粘贴到笔记中，点击即可打开资源管理器对应目录。

**PDF 嵌入：**将 PDF 文件直接拖入笔记，会自动显示嵌入视图，不占版面，可翻页且不影响笔记布局，左侧还有缩略图导航。

![[笔记同步助手/images/fdc253fc88c11e5bc1b6a656c0f8c494_MD5.jpg|PDF 嵌入效果]]

PDF 嵌入效果
