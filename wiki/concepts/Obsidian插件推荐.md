# Obsidian 插件推荐

> 一句话：按场景分类的精选插件清单，从必装到进阶，附带每款的实际作用。

## 🔥 必装五件套（每个用户都应先装）

| 插件 | 作用 | 为什么必装 |
|------|------|------------|
| **Dataview** | 把笔记库变成可查询的数据库。通过简单代码自动生成动态表格、列表、日历 | 这是 Obsidian 的"Excel 公式"级能力——没有它，你的笔记只是文件；有了它，笔记是数据库 |
| **Templater** | 高级模板引擎，支持动态日期、JS 逻辑、光标定位 | 比内置模板强大 10 倍，是构建自动化写作流的核心 |
| **Omnisearch** | 更强全局搜索，支持模糊搜索、OCR 图片文字、PDF 全文 | Obsidian 内置搜索够用但不好用，装了这个就再也不愁找不到笔记 |
| **Tasks** | Markdown 任务管理，支持日期/优先级/循环任务/跨笔记聚合 | 替代 Todoist，和笔记深度绑定——任务就是笔记的一部分 |
| **Obsidian Git** | 自动 Git 版本管理 + 远程备份 | 免费、自动、可回退——最安心的备份方案 |

## 🤖 AI 智能助手（2025-2026 最热门）

| 插件 | 作用 | 推荐理由 |
|------|------|----------|
| **Claudian** | 🔥 **已装**。Claude Code 嵌入侧边栏，选中文字可改写/总结/翻译，基于整个知识库回答问题 | 。Claude 原生集成，Claude Code 用户首选 |
| **Copilot for Obsidian** | 全库 AI 对话，支持 Agent 模式。免费（自带 Key）或 Plus 订阅 | 最稳定的 AI 聊天插件，GitHub 5776⭐ |
| **Smart Connections** | AI 智能关联推荐，输入时自动发现笔记间的潜在链接 | 理解笔记的"语义灵魂"，比手动翻找高效得多 |
| **Text Generator** | 开源 AI 文本生成，支持 OpenAI/Anthropic/Ollama 等 | 续写、翻译、摘要，一个插件全搞定 |
| **AI Tagger** | 一键分析笔记内容，自动打标签 | 大型知识库组织利器 |

## 📋 任务与项目管理

| 插件 | 作用 |
|------|------|
| **Kanban** | 看板视图，把笔记变成待办/进行中/已完成三列，拖拽管理 |
| **QuickAdd** | 快速捕获——一键创建笔记、追加到日记、或运行多个操作的宏 |
| **Calendar** | 日历侧边栏，点击日期创建/打开日记 |
| **Periodic Notes** | 日记/周记/月记系统，自动生成周期性笔记 |

## ✍️ 写作与输出

| 插件 | 作用 |
|------|------|
| **Excalidraw** | 手绘风格图表/流程图/思维导图，可嵌入笔记 |
| **Linter** | 自动格式化 Markdown（YAML 对齐、标题层级、空行修复）|
| **MP Preview** | 公众号排版预览，一键复制格式化文章 |
| **Various Complements** | 输入时自动补全双向链接和关键词，降低链接阻力 |

## 🎨 界面与体验

| 插件 | 作用 |
|------|------|
| **Make.md** | 文件夹加图标、封面背景、自定义工作区——让 Obsidian 更像 Notion |
| **Iconize** | 给文件/文件夹添加 emoji 图标，视觉快速定位 |
| **Hover Editor** | 悬浮预览时可直接编辑笔记，不用切换标签页 |
| **Editing Toolbar** | Word 风格编辑工具栏，降低上手成本 |
| **Banners** | 为笔记添加头图，视觉体验拉满 |

## 🔧 维护与安全

| 插件 | 作用 |
|------|------|
| **BRAT** | 🔥 **已装**。安装未上架官方市场的 Beta 插件 |
| **Clear Unused Images** | 扫描一键清理未被引用的图片 |
| **Janitor** | 清理无用文件和空文件夹 |
| **Find Orphaned Files** | 找出无链接无引用的孤立笔记 |

## 🎯 按使用场景推荐

### 你的场景：政务项目经理 + 笔记控
> **核心**：Dataview + Tasks + Templater + Omnisearch + Obsidian Git
> **AI**：Claudian（已有）+ Smart Connections（推荐加）
> **维护**：Linter + Clear Unused Images
> **视觉**：Make.md + Iconize

### 学生/研究者
> HoverNotes + Spaced Repetition + Dataview + Excalidraw + Omnisearch

### 内容创作者
> Claudian + MP Preview + Tasks + Excalidraw + Various Complements

### 极简入门
> Dataview + Tasks + Omnisearch + Linter + Obsidian Git

## ⚠️ 避坑建议

1. **一次装 1-2 个**，用熟再加，不要一次装 20 个
2. **从需求出发** — 先问"我缺什么功能"，再去找插件
3. **定期清理** — 3 个月没用过的就卸载
4. **别追求数量** — 真正核心的通常不超过 10 个
5. **注意冲突** — 不要装多个功能重复的插件

> 插件是手段，不是目的——让 Obsidian 真正变成「第二大脑」，而不是「插件展示柜」。
