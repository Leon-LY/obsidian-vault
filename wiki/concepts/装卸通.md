# 装卸通（装卸通信息化监管系统）

> 一句话：依托"爱山东+山东通"双端架构的渔港装卸行业数字化监管系统，善思明科技承建。

## 概述

装卸通信息化监管系统由 [[荣成市交通运输局]] 发起建设，**山东善思明科技发展股份有限公司**（我方公司）承建。系统依托"威海市基层数据服务平台"，采用 **爱山东（移动端）+ 山东通（PC端）** 双端架构，解决荣成市渔港码头装卸行业的监管痛点。

荣成市作为重要沿海港口城市和水产品冷链集散地，装卸行业存在底数不清、合同缺失、考勤混乱、监管困难、劳资纠纷频发等突出问题。

## 系统架构

```
爱山东 APP（移动端）← → 山东通（PC端后台）
         ↓                        ↓
  务工人员/装卸队/        镇街/部门监管人员
  用工单位/监管人员

后端: Java Spring Boot · 数据库: MySQL · 加密: SM2国密
```

## 四类角色移动端功能

### 1. 务工人员端
- **上班/下班打卡**: 调用爱山东定位 API，自动关联合同期内项目
- **个人工作量查看**: 计时/计件数据透明可查
- **合同查阅**: 在线查看劳动合同
- **投诉反馈**: 投诉装卸队或用工单位违规情况

### 2. 装卸队端
- **项目新建**: 开工前录入 20+ 字段的项目信息
- **用工人员登记**: 身份证去重校验，系统已有直接选择
- **作业登记**: 计时自动统计 / 计件手动录入
- **车辆管理**: 常用车牌提前录入

### 3. 用工单位端
- **作业审核**: 确认装卸队提交的工作备案
- **人员/车辆查询**: 查看本单位所有工作人员和车辆

### 4. 监管人员端
- **备案终审**: 镇街最终确认审核
- **鲁通码扫描核查**: 扫码核实身份（零额外费用）
- **身份证号查询核查**: 手动输入匹配
- **预警信息**: 未缴社保/医保、未签合同人员清单
- **投诉管理**: 协助处理维权

## PC 端后台（山东通）

镇街/部门监管人员统一管理：
- 务工人员信息管理（含禁用标识）
- 装卸队信息管理（含红黑名单三级管理）
- 备案管理（注册地+用工地双重审核）
- 核查记录管理
- 红黑名单审核公示
- 工作量统计（支持 Excel 导出）
- 培训资料管理

## 红黑名单三级管理

| 等级 | 说明 |
|------|------|
| 红名单一级 | 信用良好，优先推荐 |
| 红名单二级 | 一般守信 |
| 黑名单三级 | 违规失信，全城共享 |
