检查知识库健康状况。

## 检查项

### 1. 断链
哪些 [[wikilink]] 指向不存在的页面？
列出每个断链所在文件和目标。

### 2. 孤立节点
哪些知识库页面入链 + 出链 < 2？
——这些可能是"写了但从未被引用"的孤岛，考虑合并或删除。

### 3. 矛盾检测
有没有两个页面说了相反的事但没有标记「⚠️ 待确认」？

### 4. 过时内容
有没有超过 3 个月未更新 `updated` 字段的页面？
列出，询问我是否需要刷新。

### 5. 目录一致性
- 知识库/目录.md 列出的页面是否都存在？
- 是否有没有被目录.md 收录的知识库页面？

### 6. 标签规范
有没有不符合 CLAUDE.md 标签体系的标签？
列出不规范标签和所在页面。

### 7. 知识盲区
CLAUDE.md 里关注的领域（智慧城市、政务信息化、招投标……），有没有该有但缺失的概念页面？

---

## 输出格式

```
📊 知识库健康报告 | YYYY-MM-DD

🔴 严重（需立即处理）：N 项
🟡 建议（本周处理）：N 项
🟢 良好：N 项

### 详情
（按严重程度排序，每项标注所在文件和修复建议）
```
