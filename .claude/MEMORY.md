**铁律 1 · 项目名称**：产品正式名称为 **「并蒂星球」**（原名"双生星球"，2026-06-12 已完成 vault 全量迁移）。代码仓库可能仍含旧名 `twin-planet`，但 Obsidian vault 内所有文档、设计稿、产品描述、commit message 一律使用「并蒂星球」。意象来源：并蒂莲——一茎两花、同根共生。

**铁律 2 · 颜色分配（V3）**：**同色系不同明度**。大宝=深陶土 `#A45C40`，二宝=浅陶土 `#C7866A`。按出生顺序分配，不按性别。核心原因：蓝粉组合会在用户潜意识里制造"蓝=男、粉=女"的性别暗示——两个男孩的家庭看到粉色会不适。新方案无论什么性别组合都适用。**禁止**在任何 UI 中使用蓝粉二元对立。
